# USTC AGI · Table Mining Homepage

This repository hosts the static website for the USTC AGI Table Mining Group, detailed in https://ustcagi-table-mining.github.io.  
It showcases our research pillars, flagship papers, and resources that span benchmarks, reasoning frameworks, and domain-specific datasets.

## Local preview

```bash
python3 -m http.server 4173
```

Navigate to <http://localhost:4173> and you should see the landing page served from `index.html`.

## Structure

- `index.html` — single-page site with hero, research focus areas, highlights, and resources.
- `styles.css` — minimal dark-theme styling with responsive layout.

Update any text or links directly inside `index.html`. Styles can be adjusted through CSS custom properties at the top of `styles.css`.
