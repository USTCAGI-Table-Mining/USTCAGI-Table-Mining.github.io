2026.03

TabClaw 开源发布，面向个性化、自进化的本地交互式表格分析 AI 智能体。 TabClaw is open-sourced, introducing a personalized, self-evolving local AI agent for interactive table analysis.

TabClaw 支持 CSV/Excel 输入与自然语言目标，融合计划先行、并行智能体、记忆与技能蒸馏机制，实现全流程交互式表格分析。 TabClaw takes CSV/Excel inputs with natural language goals, combining plan-before-act, parallel agents, memory, and skill distillation for end-to-end interactive table analytics.

GitHub Project
2026.03

TableMind++ 预印本发布，提出不确定性感知的程序化智能体，用于工具增强的表格推理。 TableMind++ preprint released, introducing an uncertainty-aware programmatic agent for tool-augmented table reasoning.

TableMind++ 在 TableMind 基础上引入记忆引导计划剪枝、置信度动作精化与双权重轨迹聚合，有效缓解 LLM 幻觉问题。 TableMind++ extends TableMind with memory-guided plan pruning, confidence-based action refinement, and dual-weighted trajectory aggregation to mitigate LLM hallucinations.

该工作利用不确定性感知推理机制，显著提升了表格多跳推理的稳定性与可解释性。 This work leverages uncertainty-aware inference to build transparent and stable reasoning trajectories over structured tables.

研究方向 Research Areas

01

表格结构识别 Table Structure Recognition

面向复杂表格图像的结构解析，构建层次化评测基准与视觉工具链，提升低质量输入的识别鲁棒性。 Structure parsing for complex table images with hierarchical benchmarks and visual toolchains to improve robustness on low-quality inputs.

02

时序表格智能 Temporal Table Intelligence

将多变量时间序列重构为文本表格，利用大语言模型实现零样本时序分类与推理。 Reformulating multivariate time series as textual tables to enable training-free LLM-based classification and zero-shot reasoning.

03

慢思考表格推理 Slow-Thinking Table Reasoning

基于不确定性感知的慢思考推理机制，构建可解释、稳定的多跳表格推理轨迹。 Uncertainty-aware slow-thinking LLMs that create transparent and stable multi-hop table reasoning trajectories.

04

表格分析智能体 Table Analysis Agents

融合计划–执行范式、工具调用与沙箱执行的自主表格分析智能体，实现可信赖的结构化数据分析。 Autonomous agents combining plan-then-execute reasoning, tool augmentation, and sandboxed execution for trustworthy table analytics.

05

领域表格基准 Domain Table Benchmarks

面向化学、科学报告等特定领域，构建专业多模态表格评测数据集，推动跨领域表格理解研究。 Expert-curated multimodal table benchmarks for chemical and scientific domains to advance cross-domain table understanding.

06

综述与基础研究 Survey & Foundation

系统梳理大语言模型驱动的表格挖掘研究进展，识别挑战与未来机遇，为领域发展提供参考。 Comprehensive reviews of LLM-driven table mining, surfacing open challenges and future opportunities for the community.
